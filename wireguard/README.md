# WireGuard VPN Configuration

Эта директория содержит шаблоны конфигурации для настройки VPN между сервером Hetzner и офисной сетью.

## Структура

- `wg0-server.conf.template` - Шаблон конфигурации для сервера (Hetzner)
- `wg0-client.conf.template` - Шаблон конфигурации для клиента (Keenetic Router)
- `generate-keys.sh` - Скрипт генерации ключей

## Использование

1. Запустите скрипт генерации ключей:
   ```bash
   cd wireguard
   bash generate-keys.sh
   ```

2. Скрипт создаст:
   - Приватные и публичные ключи для сервера и клиента
   - Готовые конфигурационные файлы с подставленными ключами

3. Примените конфигурации:
   - На сервере: скопируйте `wg0-server.conf` в `/etc/wireguard/`
   - Для Keenetic: импортируйте содержимое `wg0-client.conf`

## Безопасность

⚠️ **ВАЖНО**: Приватные ключи (.key файлы) не должны попадать в Git!
Они добавлены в `.gitignore`.
